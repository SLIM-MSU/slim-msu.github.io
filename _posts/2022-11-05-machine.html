Time Projection Chambers (TPCs) produce high-dimensional, sparsely populated datasets that encode detailed three-dimensional particle tracks. This project develops machine learning methods that learn structured representations directly from raw detector signals, enabling scalable analysis and discovery in nuclear physics experiments.

We focus on sparse convolutional neural networks that operate directly on TPC data represented as sparse tensors. Using Minkowski Engine architectures, we investigate how latent embeddings learned from detector signals capture physically meaningful event structure and transfer across different TPC systems, demonstrating a pathway toward foundation models for TPCs.

This effort builds on earlier work developing a multi-modal deep learning framework for the GADGET II TPC at the Facility for Rare Isotope Beams (FRIB), where convolutional neural network object detection was combined with time-projection signal analysis to identify rare proton–alpha coincidence events relevant to nuclear astrophysics measurements.
